MelPharma WebApp

    PostgreSQL 16

Backend:
    Python

API:
    RestAPI

Dependencies:
    Flask, flask_restful
    SQLAlchemy
    psycopg2

UI:
    HTML5
    CSS - Bootstrap, Flexbox
    JavaScript
